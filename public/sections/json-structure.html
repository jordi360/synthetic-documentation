JSON Files

JSON files play a crucial role in making our scene setup highly customizable and efficient. These files contain detailed configurations for various components of our synthetic environment, allowing for dynamic and flexible scene creation.

machines.json

Machine configurations specify the .blend file for the machine, camera intrinsics and predefined camera positions. This enables the setup of various camera angles to capture the synthetic scenes from different perspectives.

belt.json

The configuration for conveyor belts specifies the .blend file to use, the distance between belt pieces, the total length of the belt and its height, as well as optional color settings.

bottles.json

For bottles, the JSON configuration includes the object name, the .blend file to use, the collection name and a transformation matrix to position the product in the scene. It also references a separate JSON file for the bottle's labels.

labels.json

The files for product labels contain information about the label image paths, their desired width, positioning (latitude and longitude), rotation and any anomalies such as rotation, wrinkles, scratches and stains.
